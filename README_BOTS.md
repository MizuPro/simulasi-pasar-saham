# Smart Bot Simulation for M-BIT Platform

This folder contains Python scripts to simulate realistic market activity using "Smart Bots".

## Files

1.  `bot_setup.py`: Script to register, initialize, and fund the bots.
2.  `smart_bot.py`: The main script that runs the simulation logic (Trading, FOMO, Panic, Bandar games).
3.  `bot_data.json`: Generated by `bot_setup.py`, contains credentials for the bots (do not edit manually).

## Prerequisites

1.  **Python 3.x**
2.  **Dependencies:**
    ```bash
    pip install requests
    ```
3.  **Running Backend:** The M-BIT Platform backend must be running (`localhost:3000`).
4.  **Admin Account:** An admin account must exist.
    *   Default assumed: `admin` / `admin123`
    *   If different, update `ADMIN_USERNAME` and `ADMIN_PASSWORD` in `bot_setup.py`.

## Usage

### 1. Setup Bots
Run this once to create the bot accounts and give them money (RDN) and shares.

```bash
python3 bot_setup.py
```

*   **Configuration:** Open `bot_setup.py` to adjust:
    *   `NUM_RETAIL_BOTS`: Number of retail bots (default: 10).
    *   `NUM_BANDAR_BOTS`: Number of bandar bots (default: 2).
    *   `RETAIL_RDN_...`: Balance range for retail.
    *   `BANDAR_RDN_...`: Balance range for bandar.

### 2. Run Simulation
Run this to start the trading activity.

```bash
python3 smart_bot.py
```

*   **Behavior:**
    *   Bots will login using `bot_data.json`.
    *   They will randomly select active stocks to trade.
    *   **Retail Bots:** React to price changes (FOMO/Panic) or trade randomly.
    *   **Bandar Bots:** Place large "Support" bids, or "HAKA" (Eat Offer) to push prices up.
    *   **Logs:** The script prints actions to the console (e.g., `[bot_retail_1] BUY MICH @ 1250 x 5 Lots - OK`).

### 3. Stop
Press `Ctrl+C` to stop the simulation.

## Strategies Implemented

*   **Retail:**
    *   **Normal:** Random small trades to create liquidity.
    *   **FOMO:** Buys aggressively if price rises > 2%.
    *   **Panic:** Sells aggressively if price drops > 2%.
*   **Bandar:**
    *   **Accumulation:** Places thick buy walls (Support) below current price.
    *   **Mark Up:** Buys large volume at best offer (HAKA).
    *   **Dump:** Sells large volume at best bid (HAKI).
